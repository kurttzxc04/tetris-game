# Tetris Game

A simple **Tetris game** built using **HTML, CSS, and JavaScript**.

## Features
- Classic Tetris gameplay
- Easy & Normal difficulty modes
- Next block preview (Easy mode)
- Dark / Light mode toggle
- High score tracking
- Pause and restart options

## Technologies Used
- HTML
- CSS
- JavaScript
- HTML Canvas

## How to Run
1. Download the project
2. Open `index.html` in a browser
3. Play the game

## Controls
- Arrow keys: Move blocks
- Space: Hard drop
- Q / E: Rotate
- P: Pause
- R: Restart
